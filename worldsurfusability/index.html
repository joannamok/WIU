WorldSurf Usability Test Page

The goal of the Surf World website is to create a user-friendly experience for our customers. The new interface is designed to be easy to learn and time efficient for accomplishing tasks. The intent is that Surf World will be a memorable website and motivate return customers and future recommendations.

This webpage has been set up to document usability tests. They have been tested on real users to so we can better understand the full experience of visitors to the Surf World website and any need for possible improvements.
